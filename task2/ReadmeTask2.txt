## 📖 Welcome! What is this project?

Welcome to our Group 30 self-study project! In this repository, we explore the transition from linear data structures (like Stacks and Queues) to non-linear **Graph Abstract Data Types (ADT)**. 

Specifically, we tackle the **Single-Source Shortest Path (SSSP)** problem using **Dijkstra’s Algorithm**. Our project proves how mathematical "greedy" logic, combined with highly optimized data structures (like Adjacency Lists and Min-Heaps), can instantly solve massive real-world navigation problems like Google Maps routing or internet traffic protocols.

## ⚙️ How to Run the Python Code

We have designed our Python implementation to be simple, standalone, and easy to run. You do not need to install any external libraries (we only use Python's built-in `heapq` module).

### Prerequisites
* You must have **Python 3.x** installed on your computer.

### Step-by-Step Instructions
1. **Download the code:** Clone this repository or download the `dijkstra_graph.py` file to your local machine.
2. **Open your terminal:** Open Command Prompt (Windows), Terminal (Mac/Linux), or the terminal built into your IDE (like VS Code).
3. **Navigate to the folder:** Use the `cd` command to go to the folder where you saved the file.
4. **Run the script:** Type the following command and press Enter:
   ```bash
   python dijkstra_graph.py
